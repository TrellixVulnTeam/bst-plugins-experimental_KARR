tar_element plugin¶

tar_element - Output tarballs¶

An element plugin for creating tarballs from the specified dependencies

Default Configuration¶

The tarball_element default configuration:
# tar_element default configurations

config:
  
  # Optionally specify a filename for the archive
  filename: files

  # Optionally specify a compression algorithm to use
  # Currently supported options:
  #   * none
  #   * gzip (default)
  #   * xz
  #   * bzip2
  compression: gzip
