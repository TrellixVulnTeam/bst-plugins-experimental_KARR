bazelize plugin¶

Bazelize kind Buildstream element plugin

It creates BUILD files calling bazel [cc_* rules](https://docs.bazel.build/versions/master/be/c-cpp.html).

As an example considering an element makelib.bst producing an artifact containing: * usr/lib/lib.so * usr/include/hdr.h

An element of this kind (‘bazelize.bst’) declaring a build-depends: makelib.bst will produce a BUILD file containing:

package(default_visibility = ["//visibility:public"])
load("@rules_cc//cc:defs.bzl", "cc_library")

cc_library(
    name = "makelib",
    srcs = ['usr/lib/lib.so'],
    hdrs = ['usr/include/hdr.h']
)
